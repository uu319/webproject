This is my php project using html, bootstrap, javascript and php.
Framework used is codeigniter.

Tis is a grading system and has 3 users(admin, student, and teacher).

*Admin can:
1. Add student, teacher.
2. Create new subjects sections and class.
3. View all classes, sections and logs.

*Student can:
1. View profile.
2. Change password.
3. View Grades.

*Teacher can:
1. View classes.
2. View students from classes.
3. Add grades to students.
